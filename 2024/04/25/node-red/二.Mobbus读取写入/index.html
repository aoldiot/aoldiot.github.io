modbus读寄存器

03 读保持寄存器

创建Modbus-Read 及 Debug节点

配置Modbus-Read节点

  • server配置

名称 说明
Type 选择通讯方式，这里选TCP
Host Tcp服务端IP地址
端口 默认502
Tcp Type DEFAULT

sever配置没问题的话，节点状态会变为connected状态

  • 其他参数配置

名称 说明
Unit-ld 485地址
FC 选择Modbus 功能码
Address 寄存器地址，本人测试的是 00001 寄存器
Quantity 指定要读取或写入的寄存器或线圈的数量
Poll Rate 轮循时间
